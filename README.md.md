PROJECT: SALES DATA ANALYSIS

Sales Performance Analysis Dashboard

An end-to-end Data Analyst portfolio project built using Python, SQL, and Power BI to simulate a real-world business analytics workflow.


---

🚀 Project Overview

This project focuses on analyzing sales data to uncover:

Revenue trends over time

Region-wise performance

Product category contribution

Discount vs sales relationship

Profitability insights

Customer and channel analysis


The workflow follows a real industry pipeline:

CSV Data → Python EDA → SQL KPI Queries → Power BI Dashboard → Business Insights
